## APIs

| 参数      | 说明                   | 类型               | 默认值  |
| :-------- | :--------------------- | :----------------- | :------ |
| children  | 组件内显示的内容       | string             | -       |
| copyable  | 内容是否可以复制       | boolean            | true    |
| language  | 组件内容的语言         | string             | 'tsx'   |
| spotlight | 是否添加聚光灯背景效果 | boolean            | false   |
| symbol    | 组件内容前显示的符号   | string             | -       |
| type      | 组件的渲染类型         | 'ghost' \| 'block' | 'ghost' |
